# Security Policy

## Supported Versions

Only the latest release version is supported with security updates:

| Version | Supported |
| :--- | :---: |
| v1.0.0 |  Yes |
| < v1.0.0 |  No |

## Reporting a Vulnerability

We take the security of FastCure seriously. If you find a security issue or vulnerability, please do NOT create a public GitHub Issue. Instead, follow these steps:

1.  **Draft Report**: Document the exact vulnerability details, E2E steps to reproduce, and possible impact.
2.  **Email Us**: Send your report directly to `security@fastcure.com` (use GPG encrypting if sensitive).
3.  **Review**: Our response team will acknowledge your report within 48 hours and work with you to analyze and patch the vulnerability under a responsible disclosure policy.
